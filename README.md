# Grovia — Premium Grocery MERN Application

Grovia is a modern premium grocery e-commerce web application built using the MERN stack.  
The project focuses on elegant UI design, glassmorphism aesthetics, responsive layouts, authentication, cart management, wishlist functionality, and a luxury grocery shopping experience.

---

# Features

- Premium modern grocery UI
- Responsive design for all devices
- JWT Authentication
- Login & Signup System
- MongoDB Database Integration
- Add To Cart Functionality
- Wishlist Functionality
- Quantity Management
- Place Order System
- Cash On Delivery Popup
- Product Search Bar
- Product Filtering
- Animated Grocery Elements
- Profile Dropdown
- Premium Membership Card
- Order Count Section
- Smooth Hover & Click Animations
- Glassmorphism Design
- MERN Stack Architecture

# Installation

## Clone Repository

```bash
git clone YOUR_GITHUB_REPO_LINK
```

---

# Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

# Backend Setup

```bash
cd server
npm install
npm run dev
```

---

# Environment Variables

Create `.env` inside the server folder:

```env
MONGO_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET_KEY
```
